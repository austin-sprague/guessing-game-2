Code Fellows Day One Guessing Game

Javascript on this page will be for a guessing game.

A guessing game?

Yes - a three question guessing game about me!

How does it work?

Step One: The Prompt

The first step asks for the user's name and then saves that into a variable for later reference. There will also be a concatenated response that greets the user using the new variable.

Step Two: The Three Questions

There will be three questions about me that will be prompted for the user to answer with a Yes or No response. The answer will be saved as variable question1, question2 and question3.

The user response will be logged into the console.

Step Three: Analysing the Answers

After each question, there will be an if/else statement that will analyse whether the user answered the question correctly.

The if statement will check whether the user's answer equates the correct string. There will be an alert of an immediate feedback to their answer.

Included is error correction that will turn all the user input into capital letters using the method of .toUpperCase() and will also accept the input of y or n into the prompt.

Whether the answer is correct or incorrect will be logged into the console.

Step Four: Feedback

If the user answered correctly, an alert will display a specific and personalised confirmation message. If answered incorrectly, there will be an alert that displays the correct answer in a personalised and specific message.

Step Five: Summary

The final step gives the user a summary of the answers they submitted correctly. It then thanks them for playing the game.
