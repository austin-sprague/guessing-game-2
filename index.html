Code Fellows Day One Guessing Game

Javascript on this page will be for a guessing game.

A guessing game?

Yes - a three question guessing game about me!

How does it work?

Step One: The Prompt

The first step asks for the user's name and then saves that into a variable for later reference. There will also be a concatenated response that greets the user using the new variable.

Step Two: The Three Questions

There will be three questions about me that will be prompted for the user to answer with a Yes or No response. The answer will be saved as variable question1, question2 and question3.

There will be an alert that reminds the user of the answers that they had submitted.

Step Three: Analysing the Answers

Once the user has answered the three prompts/questions, there will be three if/else statements that will analyse whether the user answered the questions correctly.

The if statement will check whether the user answer equates the correct string.

Step Four: Feedback

If incorrect, there will be an else statement that returns to the user that the response was incorrect.
